How to Use This Guide

Three options per artifact — pick whichever works best for you.

Option A — Bookmarklet
Drag & Click
Drag the colored button to your bookmarks bar. Navigate to app.people.ai, click the bookmark. Table creates and opens automatically.
Option B — Console Script
Copy & Paste
Expand the console script, copy it, open DevTools on app.people.ai (Cmd+Option+J / Ctrl+Shift+J), paste into the Console tab and press Enter.
After Each Deploy
Record the ID
Each script logs the created table ID. Paste it into the ID field below the card — this feeds the board generator in Step 2.
