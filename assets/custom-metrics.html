Group 1 Base Metrics 7 metrics
Email Responsiveness
account_email_responsiveness
FormulaMetric
Formula
a / b
Data Type
double
Description
Ratio of inbound to outbound emails at the account level. Parameter a = inbound emails (external, non-outbound). Parameter b = outbound emails (external, outbound).
Time Ranges
last_7_days, last_14_days, last_30_days, this_quarter, any_time
Coaching: Above 0.5 = two-way conversation. Below 0.3 = one-way prospecting.
T1: ICP Fit T2: Relationship Health T5: Account Neglect
Executive Coverage
account_executive_coverage
FormulaMetric
Formula
100 * a / b
Data Type
percentage
Description
% of external activities involving Director, VP, or Executive-level contacts. Parameter a = exec-level activities. Parameter b = all external activities.
Time Ranges
last_14_days, last_30_days, this_quarter, any_time
Coaching: Above 30% with no open opp = strategic conversation needs a deal.
T1: ICP Fit T2: Relationship Health
Engagement Momentum
account_engagement_momentum
FormulaMetric
Formula
100 * a / b
Data Type
percentage
Description
Rate of engagement acceleration. Parameter a = meetings + inbound emails (14d). Parameter b = meetings + inbound emails (30d). Compares recent vs trailing activity.
Time Ranges
last_14_days (a) vs last_30_days (b)
Coaching: Above 100% = engagement accelerating. 50-100% = steady. Below 50% = engagement fading.
T3: Engagement Momentum T4: Whitespace
Stakeholder Breadth
account_stakeholder_breadth
FormulaMetric
Formula
a / 4.3
Data Type
double
Description
Unique external contacts engaged per week (30-day average). Parameter a = count_distinct of external participant emails over 30 days, divided by 4.3 weeks.
Time Ranges
last_30_days
Coaching: Below 1.0/week = thin relationship. 1.0-3.0 = healthy. Above 3.0 = broad multi-threaded engagement.
T1: ICP Fit T2: Relationship Health
Meeting Acceleration
account_meeting_acceleration
FormulaMetric
Formula
4.286 * a / b
Data Type
double
Description
Ratio of 7-day to 30-day meeting frequency, normalized (4.286 = 30/7). Above 1.0 = meetings accelerating. Parameter a = meetings (7d). Parameter b = meetings (30d).
Time Ranges
last_7_days (a) vs last_30_days (b)
Coaching: Above 1.5 = surge. 0.8-1.2 = steady. Below 0.5 = meeting cadence declining.
T3: Engagement Momentum
Email Acceleration
account_email_acceleration
FormulaMetric
Formula
4.286 * a / b
Data Type
double
Description
Ratio of 7-day to 30-day inbound email frequency, normalized. Above 1.0 = inbound email accelerating. Parameter a = inbound emails (7d). Parameter b = inbound emails (30d).
Time Ranges
last_7_days (a) vs last_30_days (b)
Coaching: Above 1.5 = surge in inbound interest. 0.8-1.2 = steady. Below 0.5 = going quiet.
T3: Engagement Momentum
Activity Intensity
account_activity_revenue_ratio
FormulaMetric
Formula
a + b
Data Type
double
Description
Total external meetings + emails in last 30 days. Used as a proxy for activity-to-revenue comparison. Revenue normalization done in the SalesAI signal layer.
Time Ranges
last_30_days
Coaching: Compare against account revenue tier. High intensity on small accounts = expansion signal. Low intensity on large accounts = coverage gap.

Group 2 Composite Scores 5 scores
ICP Fit Score
account_icp_fit_score
Composite
Components
engagement_level + executive_coverage + email_responsiveness + stakeholder_breadth
Weighting
Engagement 30%, Exec Coverage 25%, Email Responsiveness 25%, Stakeholder Breadth 20%
Coaching: Weighted blend that identifies accounts behaving like your best customers. Score 0-100. Above 70 = strong ICP fit candidate.
T1: ICP Fit Profiling
Relationship Health Score
account_relationship_health_score
Composite
Components
stakeholder_breadth + executive_coverage + email_responsiveness + meeting_frequency
Weighting
Stakeholder Breadth 30%, Exec Coverage 25%, Email Responsiveness 25%, Meeting Frequency 20%
Coaching: Multi-threading and depth assessment. Score 0-100. Below 30 = single-threaded risk. Below 20 = critical — one contact leaves and you lose the account.
T2: Relationship Health
Momentum Score
account_momentum_score
Composite
Components
engagement_momentum + meeting_acceleration + email_acceleration
Weighting
Engagement Momentum 40%, Meeting Acceleration 35%, Email Acceleration 25%
Coaching: Trajectory indicator. Above 1.0 = account heating up. Below 0.5 = cooling down. 0.8-1.2 = steady state.
T3: Engagement Momentum
Expansion Potential Score
account_expansion_score
Composite
Components
engagement_level + activity_intensity + stakeholder_breadth + executive_coverage
Weighting
Engagement 25%, Activity Intensity 30%, Stakeholder Breadth 25%, Exec Coverage 20%
Coaching: Identifies accounts where engagement intensity exceeds current deal footprint. High score + Customer account type + few open opps = expansion opportunity.
T4: Whitespace & Expansion
Neglect Risk Score
account_neglect_risk_score
Composite
Components
engagement_level (inverted) + activity_intensity (inverted) + meeting_acceleration (inverted)
Weighting
Engagement (inv) 40%, Activity Intensity (inv) 35%, Meeting Acceleration (inv) 25%
Coaching: Higher score = more neglected. Inverted metrics mean low engagement drives high neglect risk. Prioritize accounts with high neglect score AND high annual revenue.
T5: Account Neglect
